I created a light using the ESP32 development module, which can be controlled using a mobile app.

Key Skills Showcased:
1. ESP32 Programming: Taming the versatile microcontroller to orchestrate light's every move.
2. IoT Communication: Establishing a seamless wireless bridge between the light and my smartphone, powered by WiFi connectivity.
3. Mobile App Development: Crafting an intuitive interface for effortless control, allowing me to switch moods and create ambiance with a tap.
4. Circuit Design: Engineering the electrical connections to ensure a bright and safe experience.

How It Works:
1. App-Driven Commands: From the comfort of my phone, I issue commands to adjust brightness, change colors (if applicable), or set schedules.
2. Wireless Transmission: The app sends these commands through the ether, using WiFi as a conduit.
3. ESP32 Reception: The ESP32, the light's intelligent heart, diligently receives and decodes the signals.
4. Signal Conversion: Acting as a digital translator, the ESP32 transforms commands into instructions that the light can understand.
5. Illumination Execution: The light obediently obeys, adjusting its brightness, color, or schedule according to my wishes.
